# FEM Module

This module provides Python bindings for various FEM components using Pybind11. The key classes and functions include:

- **Gradient**
- **HyperElasticPotential**
- **Jacobian**
- **Laplacian**
- **MassMatrix**
- **Mesh**
- **ShapeFunctions**
